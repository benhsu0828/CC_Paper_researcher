# concise

精煉直接，重點突出，快速閱讀

## 寫作特點

- 直擊核心，省略鋪墊
- 每段一個要點，不展開
- 用表格和列表提高資訊密度
- 適合已有背景知識的讀者

## 語言風格

- 簡短句子，避免從句巢狀
- 直接陳述結論，再給證據
- 使用"核心是"、"關鍵在於"等提示詞
- 不用比喻和故事

## 結構特點

- 一句話總結 → 核心創新 → 關鍵結果 → 侷限性
- 大量使用表格對比
- 每節控制在 2-3 段

## 示例開頭

```
**核心創新**：Engram 在 Transformer 中引入 O(1) 查表機制，
將靜態知識從計算中分離。

**關鍵設計**：
- N-gram 雜湊檢索：多頭雜湊對映到嵌入表
- 上下文門控：用隱藏狀態決定是否採納記憶
- 確定性索引：支援預取，推理開銷 <3%

**主要結果**：等引數下 MMLU +3.4，BBH +5.0
```

## 適用場景

快速瞭解、論文速覽、技術調研
