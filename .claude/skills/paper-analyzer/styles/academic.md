# academic

專業嚴謹，術語準確，適合學術交流

## 寫作特點

- 使用標準學術術語，不做過度簡化
- 準確引用論文中的定義和公式
- 客觀陳述，減少主觀評價
- 注重方法論和實驗設計的完整性

## 語言風格

- 正式書面語
- 使用"本文提出"、"實驗表明"等學術表達
- 保留原文關鍵術語（中英對照）
- 避免口語化和比喻

## 結構特點

- 摘要 → 背景與動機 → 方法 → 實驗 → 分析 → 結論
- 每個章節有明確的技術目標
- 強調與相關工作的對比

## 示例開頭

```
本文提出 Engram，一種基於條件記憶（Conditional Memory）的稀疏架構。
該方法將 N-gram 查表機制與 Transformer 主幹網路相結合，
通過可擴充套件的雜湊檢索實現 O(1) 複雜度的靜態知識獲取。

與傳統 MoE（Mixture of Experts）的動態路由不同，
Engram 的檢索索引完全由輸入 token 決定，具有確定性和可預取性。
```

## 適用場景

學術報告、論文綜述、研究組分享
